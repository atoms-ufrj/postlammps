post_lammps
===========

A tool for performing post-processing with lammps log files


Installation
------------

make<br>
make install<br>


Usage
-----

post_lammps action [args] [file name] [property 1] [property 2] ...<br>
  action = block or extract or histogram<br>
    block args = none<br>
    extract args = skip<br>
    histogram args = nbins<br>
